modified_files

Get all the files that have been modified in the given scope. If the scope is Scope::All returns None as the list of files in ModifiedContent. For Scope::Commits and Scope::DiffToMain the returned files include tracked files with unstaged changes, but do not include new files that are not tracked (i.e., not added to git yet). For Scope::Commits(N), a positive number of commits must be given. In case of zero or a negative number, only the last commit will be considered for finding the modified files.
