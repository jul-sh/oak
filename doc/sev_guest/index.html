Rust wrappers for instructions and structs for use by SNP-active guests to manage page state and interact with the hypervisor and the secure processor.

Modules

cpuid

This module contains structs that can be used to interpret the contents of the CPUID page that is provisioned into the VM guest memory during SEV-SNP startup.

ghcb

This module contains an implementation of the guest-hypervisor communications block (GHCB) page that can be used for communicating with the hypervisor.

instructions

Rust instruction wrappers for managing page state and interacting with the hypervisor.

msr

Rust implementations of the AMD SEV-SNP GHCB MSR protocol.

secrets

This module contains structs that can be used to interpret the contents of the secrets page that is provisioned into the VM guest memory during SEV-SNP startup.
