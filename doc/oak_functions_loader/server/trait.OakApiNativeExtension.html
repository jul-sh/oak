fn invoke(&mut self, request: Vec<u8>) -> Result<Vec<u8>, OakStatus>

Invokes the extension with the given request and returns a result. If no result is expected, the result is empty. An error within the extension is reflected in the OakStatus. TODO(#2701): Stop returning OakStatus for errors internal to extensions.
